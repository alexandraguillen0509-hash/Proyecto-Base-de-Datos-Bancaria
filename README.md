Banking Management Database es una base de datos diseñada para gestionar las operaciones principales de un sistema bancario, incluyendo clientes, cuentas bancarias, transacciones y préstamos.

El sistema permite registrar operaciones financieras, controlar saldos de cuentas y gestionar la relación entre clientes y sus productos bancarios.

La base de datos fue desarrollada utilizando Microsoft SQL Server.
